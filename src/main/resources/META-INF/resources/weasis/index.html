Weasis Launcher

The XNAT Weasis plugin exposes secure launch endpoints that allow you to open imaging sessions in the Weasis desktop viewer.

How to launch

  1. Select a session inside XNAT and click View in Weasis.
  2. XNAT generates a weasis:// link that passes authentication and study identifiers straight to the viewer.

Developers can also call the REST endpoint at /xapi/weasis/launch/projects/{projectId}/sessions/{sessionId}.
